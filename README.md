# EVM 批量归集与分发工具 (EVM Batch Tool)

这是一个基于 Web 的轻量级加密货币批量处理工具，专为 EVM 兼容链（如 Ethereum, Arbitrum, BSC, Optimism, Polygon 等）设计。

它是一个纯前端项目（Single Page Application），所有的逻辑都在您的浏览器本地运行，**私钥永远不会上传到任何服务器**，确保资金安全。

## 核心功能

### 1. 多种转账模式
*   **多对多 (M:N)**: 批量转账，每一行私钥对应每一行接收地址。适合点对点批量打款。
*   **归集 (M:1)**: 将多个钱包的资金批量汇集到一个主账户。适合资金归集整理。
*   **分发 (1:N)**: 使用一个主账户向多个地址批量分发资金。适合空投、发工资等场景。

### 2. 灵活的金额设置
*   **固定金额**: 发送指定的 ETH/Token 数量。
*   **百分比**: 发送账户余额的指定百分比（例如 50%）。
*   **全部数量 (MAX)**: 自动计算并扣除 Gas 费，发送账户的所有可用余额（清空账户）。
*   **随机数量**: 在指定范围内随机生成转账金额（适合交互防女巫）。

### 3. 高级控制
*   **随机延迟**: 可设置每笔交易之间的随机等待时间（秒），模拟真实用户行为。
*   **Gas 预估**: 实时连接 RPC 节点获取 Gas Price，动态计算最大可转金额，防止因 Gas 不足导致失败。
*   **ERC20 支持**: 支持原生代币（ETH/BNB/MATIC）以及任意 ERC20 代币的批量转账。

### 4. 安全与隐私
*   **本地运行**: 代码开源，无后端服务，无数据库。
*   **网络检查**: 内置 RPC 节点健康检查，显示延迟和区块高度。
*   **私钥格式兼容**: 自动处理带或不带 `0x` 前缀的私钥，自动过滤空格。

## 支持的网络

本工具基于 Ethers.js 开发，支持所有 **EVM 兼容网络**：
*   Ethereum Mainnet (ETH)
*   Arbitrum One (ARB)
*   Binance Smart Chain (BSC)
*   Optimism (OP)
*   Polygon (POL/MATIC)
*   Base
*   以及任何支持自定义 RPC 的 EVM 链（如 Linea, zkSync Era, Avalanche 等）。

> **注意**: 不支持 Solana, Sui, Aptos 等非 EVM 链。

## 如何使用

### 方法 1: 直接运行 (推荐)
由于是纯静态文件，您只需要在浏览器中打开 `index.html` 文件即可直接使用。

### 方法 2: 本地服务器 (预览模式)
如果您安装了 Node.js，可以使用 `npx serve` 启动本地服务器以获得更好的体验：

```bash
# 在项目根目录下打开终端
npx serve .
```
然后访问终端显示的地址（通常是 `http://localhost:3000`）。

## 操作指南

1.  **配置网络**: 在左上角选择目标网络，或输入自定义 RPC 节点地址。
2.  **选择代币**: 留空则转账原生代币（如 ETH）；输入合约地址则转账 ERC20 代币。
3.  **设置金额**: 选择转账数量模式（固定/百分比/随机/全部）。
4.  **导入数据**: 根据选择的模式（M:N, M:1, 1:N），粘贴私钥和地址。
5.  **检查余额**: 点击确认按钮，系统会自动检查每个账户的余额是否充足。
6.  **执行**: 确认无误后，点击执行，并在弹窗中再次确认。
